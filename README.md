# 🏠 New Home Inserts

A modern, full-featured static blog website for home decor, renovation, and lifestyle content. Built with HTML5, CSS3, and Vanilla JavaScript — no frameworks needed.

## 🌐 Live Site

> **[👉 View Live on GitHub Pages](https://aijazmajeed804.github.io/new-home-inserts/)**

---

## ✨ Features

- 🔍 **Live Search** — Real-time dropdown search across all articles
- 🗂️ **Category Filtering** — Browse by Decor, Kitchen, Lifestyle, Outdoor & more
- 📝 **Admin Dashboard** — Owner-only CMS panel with article publishing and editing
- 🔐 **SEO Support** — Meta title, description, keywords, OG image per article
- 📬 **Guest Submissions** — Submit guest posts with a live preview form
- 💬 **Comments System** — Reader comment support per post
- 📧 **Newsletter Sign-up** — Subscriber capture form
- 🌙 **Dark/Light Mode** — User-preferred color scheme support

## 🗂️ Project Structure

```
home-inserts/
├── index.html              ← Homepage
├── post.html               ← Single article view
├── about.html              ← About page
├── contact.html            ← Contact form
├── submit-guest-post.html  ← Guest post submission
├── login.html              ← Admin login
├── admin.html              ← Admin dashboard (protected)
├── css/
│   └── style.css           ← All styles
├── js/
│   └── app.js              ← Core application engine
└── data/
    └── posts.json          ← Default article seed data
```

## 🔑 Admin Access

- **URL**: `/admin.html` (redirects to login if unauthenticated)
- **Login page**: `/login.html`

## 🚀 Deployment

This site auto-deploys via **GitHub Actions → GitHub Pages** on every push to the `main` branch.
